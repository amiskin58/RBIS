# PRD - Live Score Layout

## Purpose

定義 Live Score 畫面的版面配置，作為後續 UI 開發的依據。

----------------------------------------

## Layout Structure

┌─────────────────────────────────────┐
│ Header                              │
├─────────────────────────────────────┤
│ Score Board                         │
├───────────────┬─────────────────────┤
│ Baseball Field│ Side Panel          │
│               │                     │
│               │                     │
├───────────────┴─────────────────────┤
│ Action Panel                        │
└─────────────────────────────────────┘

----------------------------------------

## Header

顯示：

- 比賽資訊
- 主隊
- 客隊
- 局數

----------------------------------------

## Score Board

顯示：

- 每局得分
- R
- H
- E
- Ball
- Strike
- Out

----------------------------------------

## Baseball Field

顯示：

- 球場
- 壘包
- 跑者位置

----------------------------------------

## Side Panel

顯示：

- 打者
- 投手
- Event History

----------------------------------------

## Action Panel

提供：

- Pitch
- Play
- Runner
- Substitution
- Finish Game

----------------------------------------

## Functional Requirements

- 所有區塊可即時更新
- 支援平板橫向操作
- 優先支援觸控操作

----------------------------------------

## Out of Scope

- Replay
- 統計分析
- 匯出
