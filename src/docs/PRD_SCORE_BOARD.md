# PRD - Score Board

## Purpose

Score Board 提供比賽即時資訊顯示。

所有資料皆由 Event 自動計算更新。

----------------------------------------

## Components

### Inning

顯示：

- 局數
- 上／下半局

----------------------------------------

### Team Score

每隊顯示：

- 每局得分
- R（Runs）
- H（Hits）
- E（Errors）

----------------------------------------

### Count

顯示：

- Ball
- Strike
- Out

----------------------------------------

### Base Status

顯示：

- 一壘
- 二壘
- 三壘

以圖示表示是否有跑者。

----------------------------------------

### Current Players

顯示：

- 打者
- 投手

----------------------------------------

## Functional Requirements

- 即時更新比分
- 即時更新 B / S / O
- 即時更新壘包狀態
- 即時更新打者與投手
- 所有資料由 Event 自動計算

----------------------------------------

## Out of Scope

- Event 編輯
- Replay
- Statistics
