# RBIS Core v1 Game Rules

## DH（指定打擊）

每場比賽主、客隊可分別設定：

主隊：

- 使用 DH
- 不使用 DH

客隊：

- 使用 DH
- 不使用 DH

系統不得限制主、客隊必須採用相同制度。

----------------------------------------

## 局數

支援：

- 5局
- 7局
- 9局

未來可自訂局數。

----------------------------------------

## Mercy Rule（提前結束）

可設定：

- 關閉
- 啟用

可設定：

- 提前結束局數
- 分差

例如：

5局 10分差

7局 15分差

----------------------------------------

## Tie Break（突破僵局）

可設定：

- 關閉
- 啟用

可設定開始局數。

----------------------------------------

## Time Limit（計時制）

可設定：

- 不限時間
- 限時比賽

例如：

60分鐘

90分鐘

120分鐘

----------------------------------------

## Extra Player（EP）

預留。

----------------------------------------

## Re-entry（再入場）

預留。

----------------------------------------

## Courtesy Runner（指定代跑）

預留。

----------------------------------------

## Suspended Game（保留比賽）

預留。

----------------------------------------

## Forfeit（沒收比賽）

預留。

----------------------------------------

## Rain Called Game（雨天裁定）

預留。

----------------------------------------

## Rule Version

每一場比賽皆保留當時採用之規則。

不得因系統更新而改變歷史比賽內容。
