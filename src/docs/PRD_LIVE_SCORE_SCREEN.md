# PRD - Live Score Screen

## Purpose

Live Score 為比賽進行中的主要操作畫面。

提供紀錄每一球、每一個 Play 及即時更新比賽資訊。

----------------------------------------

## Components

### Score Board

顯示：

- 局數
- 上／下半局
- 比數
- R / H / E

----------------------------------------

### Game Status

顯示：

- Ball
- Strike
- Out
- 壘包狀態
- 打者
- 投手

----------------------------------------

### Action Panel

包含：

- Pitch
- Play
- Runner
- Substitution
- Undo（如支援）

----------------------------------------

### Event History

顯示本場比賽 Event 紀錄。

----------------------------------------

## Navigation

Lineup
→ Live Score

Finish Game
→ Box Score

----------------------------------------

## Functional Requirements

- 記錄每球事件
- 記錄打擊結果
- 記錄跑壘事件
- 更新計分板
- 更新壘包狀態
- 更新 Event History
- 支援結束比賽

----------------------------------------

## Out of Scope

- 匯出
- 統計分析
- 賽季管理
