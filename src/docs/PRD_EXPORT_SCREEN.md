# PRD - Export Screen

## Purpose

Export 提供比賽資料的匯出功能。

所有匯出內容皆由 Event 自動產生。

----------------------------------------

## Components

### Game Information

顯示：

- 比賽日期
- 賽事名稱
- 主隊
- 客隊
- 最終比分

----------------------------------------

### Export Options

可選擇匯出：

- Event Log
- Box Score
- Game Summary

----------------------------------------

### File Format

支援：

- JSON
- CSV

預留：

- PDF
- Excel

----------------------------------------

### Actions

- 匯出（Export）
- 返回（Back）

----------------------------------------

## Navigation

Box Score
→ Export

Back
→ Box Score

----------------------------------------

## Functional Requirements

- 匯出 Event 資料
- 匯出 Box Score
- 匯出比賽摘要
- 支援 JSON
- 支援 CSV

----------------------------------------

## Out of Scope

- 資料編輯
- 雲端同步
- 賽季報表
