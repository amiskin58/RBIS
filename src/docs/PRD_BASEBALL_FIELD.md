# PRD - Baseball Field

## Purpose

Baseball Field 提供球場視覺化顯示與操作介面。

使用者可透過球場完成落點選擇及顯示跑者位置。

----------------------------------------

## Components

### Field Image

顯示：

- 內野
- 外野
- 壘包
- 本壘

----------------------------------------

### Runner Display

顯示：

- 一壘跑者
- 二壘跑者
- 三壘跑者

以圖示表示跑者位置。

----------------------------------------

### Hit Location

支援點擊球場位置，作為擊球落點。

----------------------------------------

### Defensive Position

顯示守備位置：

- P
- C
- 1B
- 2B
- 3B
- SS
- LF
- CF
- RF

----------------------------------------

## User Interaction

可執行：

- 點擊球場落點
- 顯示落點標記
- 顯示跑者位置
- 顯示守備位置

----------------------------------------

## Functional Requirements

- 支援觸控操作
- 支援點擊落點
- 即時更新跑者位置
- 即時更新守備位置
- 所有資料由 Event 自動更新

----------------------------------------

## Out of Scope

- Replay 動畫
- Heat Map
- Scout 分析
