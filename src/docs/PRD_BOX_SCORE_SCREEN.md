# PRD - Box Score Screen

## Purpose

Box Score 提供比賽結束後的完整統計資訊。

所有資料皆由 Event 自動計算產生。

----------------------------------------

## Components

### Game Information

顯示：

- 比賽日期
- 賽事名稱
- 主隊
- 客隊
- 最終比分

----------------------------------------

### Team Score

顯示：

- 每局得分
- R
- H
- E

----------------------------------------

### Batting Statistics

每位球員顯示：

- 打席（PA）
- 打數（AB）
- 安打（H）
- 打點（RBI）
- 得分（R）
- 保送（BB）
- 三振（SO）

----------------------------------------

### Pitching Statistics

每位投手顯示：

- 投球數
- 局數（IP）
- 安打（H）
- 保送（BB）
- 三振（SO）
- 失分（R）
- 自責分（ER）

----------------------------------------

### Fielding Statistics

每位球員顯示：

- 守備機會（TC）
- 刺殺（PO）
- 助殺（A）
- 失誤（E）

----------------------------------------

## Navigation

Replay
→ Box Score

Box Score
→ Export

----------------------------------------

## Functional Requirements

- 顯示完整 Box Score
- 顯示逐局得分
- 顯示打擊統計
- 顯示投球統計
- 顯示守備統計
- 所有資料由 Event 自動計算

----------------------------------------

## Out of Scope

- 編輯統計
- 即時紀錄
- 賽季分析
