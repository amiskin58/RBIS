# PRD - Home Screen

## Purpose

Home 為 RBIS Score 的入口畫面。

提供使用者快速開始、繼續或管理比賽。

----------------------------------------

## Components

### Header

顯示：

- RBIS Logo
- 使用者名稱
- 設定

----------------------------------------

### Main Menu

包含：

- 建立新比賽（New Game）
- 繼續比賽（Continue Game）
- 比賽列表（Game List）

----------------------------------------

### Recent Games

顯示最近建立或進行中的比賽。

每筆資料包含：

- 日期
- 主隊
- 客隊
- 比分（若已開始）
- 狀態（未開始／進行中／已結束）

----------------------------------------

## Navigation

New Game
→ Create Game

Continue Game
→ Live Score

Game List
→ Game List

Settings
→ Settings

----------------------------------------

## Functional Requirements

- 顯示最近比賽
- 支援建立新比賽
- 支援繼續未完成比賽
- 可進入比賽列表
- 可進入設定頁

----------------------------------------

## Out of Scope

- 統計分析
- Box Score
- Replay
