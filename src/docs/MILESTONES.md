# RBIS Project Milestones

## Milestone M1

名稱：

RBIS Score MVP

目標：

完成一套可於球場正式使用的即時紀錄系統。

完成條件：

- 建立比賽
- 建立先發名單
- 設定比賽規則
- Pitch 記錄
- Play 記錄
- Runner 記錄
- Substitution 記錄
- Replay
- Box Score
- 匯出比賽紀錄

狀態：

Planned
