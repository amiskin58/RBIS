# PRD - Create Game Screen

## Purpose

建立或編輯一場新的比賽。

----------------------------------------

## Components

### Basic Information

包含：

- 比賽日期
- 比賽時間
- 賽事名稱
- 場地

----------------------------------------

### Teams

設定：

- 主隊
- 客隊

----------------------------------------

### Game Rules

設定：

- 局數（5 / 7 / 9）
- DH（主隊）
- DH（客隊）
- Mercy Rule
- Tie Break

----------------------------------------

### Actions

- 儲存（Save）
- 下一步（Next）
- 取消（Cancel）

----------------------------------------

## Navigation

Home
→ Create Game

Save
→ Game List

Next
→ Lineup

Cancel
→ Home

----------------------------------------

## Functional Requirements

- 可建立新比賽
- 可編輯既有比賽
- 必填欄位驗證
- 儲存比賽資料
- 進入先發名單設定

----------------------------------------

## Out of Scope

- 即時紀錄
- Replay
- 統計分析
