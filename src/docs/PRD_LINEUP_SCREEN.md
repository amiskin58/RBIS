# PRD - Lineup Screen

## Purpose

設定本場比賽的先發名單與替補球員。

----------------------------------------

## Components

### Team Selection

顯示：

- 主隊
- 客隊

可切換編輯。

----------------------------------------

### Starting Lineup

每位先發球員包含：

- 打序（1～9）
- 守備位置
- 球員姓名

----------------------------------------

### Bench Players

顯示所有替補球員。

----------------------------------------

### Actions

- 儲存（Save）
- 開始比賽（Start Game）
- 返回（Back）

----------------------------------------

## Navigation

Create Game
→ Lineup

Start Game
→ Live Score

Back
→ Create Game

----------------------------------------

## Functional Requirements

- 可設定主隊先發名單
- 可設定客隊先發名單
- 可設定替補球員
- 可調整打序
- 可調整守備位置
- 儲存名單
- 開始比賽

----------------------------------------

## Out of Scope

- 比賽中換人
- Replay
- 統計分析
