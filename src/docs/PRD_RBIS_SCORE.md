# RBIS Score Product Requirement Document

Version：1.0

Status：Draft

---

# 1. Product Overview

RBIS Score 為 RBIS Platform 第一個產品。

提供棒球比賽即時紀錄功能。

以 Event 為核心，

自動產生：

- 即時計分
- Replay
- Box Score
- Statistics

所有資料可供 RBIS Manager 與 RBIS Scout 共用。

---

# 2. Product Goal

打造一套可於球場正式使用的棒球即時紀錄系統。

提供：

- 快速
- 正確
- 易操作

之紀錄流程。

---

# 3. Target Users

- Administrator
- Coach
- Recorder
- Player（Read Only）

---

# 4. MVP Scope

包含：

- 建立比賽
- 建立先發名單
- 設定比賽規則
- Pitch
- Play
- Runner
- Substitution
- Replay
- Box Score
- Export

---

# 5. Out of Scope

本版本不包含：

- AI
- 聊天
- 直播
- 售票
- 金流
- 社群功能

---

# 6. Success Criteria

第一次使用 RBIS Score 的紀錄員，

可於 10 分鐘內完成：

- 建立比賽
- 建立先發
- 完成一局紀錄
- 匯出 Box Score

即視為 MVP 成功。

---

# 7. Future Enhancement

未來將整合：

- RBIS Manager
- RBIS Scout

共同組成 RBIS Platform。
