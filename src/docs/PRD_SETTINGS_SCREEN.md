# PRD - Settings Screen

## Purpose

Settings 提供 RBIS Score 的系統設定功能。

----------------------------------------

## Components

### User Settings

包含：

- 使用者資訊
- 顯示名稱
- 語言（預留）

----------------------------------------

### Scoring Settings

包含：

- 預設局數
- 預設 Mercy Rule
- 預設 Tie Break
- 預設 DH（主隊）
- 預設 DH（客隊）

----------------------------------------

### Display Settings

包含：

- 深色模式（預留）
- 文字大小（預留）

----------------------------------------

### About

顯示：

- RBIS Score Version
- Build Number

----------------------------------------

### Actions

- 儲存（Save）
- 返回（Back）

----------------------------------------

## Navigation

Home
→ Settings

Back
→ Home

----------------------------------------

## Functional Requirements

- 修改系統預設設定
- 儲存使用者偏好
- 顯示系統版本資訊

----------------------------------------

## Out of Scope

- 帳號管理
- 權限管理
- 雲端同步
