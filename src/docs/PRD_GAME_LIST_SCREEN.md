# PRD - Game List Screen

## Purpose

Game List 提供所有比賽的查詢與管理功能。

----------------------------------------

## Components

### Header

顯示：

- 頁面標題
- 返回 Home

----------------------------------------

### Search Area

可依下列條件搜尋：

- 日期
- 賽事
- 主隊
- 客隊
- 比賽狀態

----------------------------------------

### Game List

每筆資料顯示：

- 日期
- 賽事名稱
- 主隊
- 客隊
- 比分
- 局數
- 狀態

----------------------------------------

### Actions

每場比賽可執行：

- 開始比賽（Start）
- 繼續比賽（Continue）
- 編輯（Edit）
- 檢視（View）
- 刪除（Delete）

----------------------------------------

## Navigation

Home
→ Game List

Start / Continue
→ Live Score

Edit
→ Create Game

View
→ Box Score

----------------------------------------

## Functional Requirements

- 顯示所有比賽
- 支援搜尋
- 支援排序
- 支援開始或繼續比賽
- 支援編輯
- 支援檢視
- 支援刪除

----------------------------------------

## Out of Scope

- 即時紀錄
- Replay
- 統計分析
